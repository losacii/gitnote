** Split new
  :sp file
  :vs file
  :Vex
  :Sex

** save, quit
  :up
  :w
  :xa   :x
  :5,+4w filename
  :q!

** open multiple
  vim -o a b c
  vim -O x y z
  vim -p a b c

** save as
  :saveas filename

** open new file, open another file
  :e!  file   :forcefully
  :e   file

  :e!   :reload forcefully
  :e


** Read only open
  vim -M file     (absolutely)
  vim -R file     (forcefully save)
